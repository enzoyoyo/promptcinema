# Batch Production System / 批量生产系统

## Core Principle / 核心原则

**Pipeline mode, not sequential mode.**

Don't finish one video start-to-finish before starting the next. Instead, batch each step across ALL videos.

不要一条从头做到尾。改为流水线模式，每个步骤一次性完成所有视频。

---

## Pipeline Steps

```
Step 1: Generate ALL base images (10 characters)
    ↓
Step 2: Generate ALL transformation images (10 before + 10 after)
    ↓
Step 3: Generate ALL video animations (10 videos)
    ↓
Step 4: Complete ALL CapCut edits (10 final videos)
    ↓
Step 5: Schedule ALL for publishing
```

---

## Weekly Schedule (3 hours → 14-21 pieces of content)

| Day | Task | Time |
|-----|------|------|
| Day 1 | Collect reference material (characters + motion + audio) | 30 min |
| Day 2 | Batch generate all images + transformations + animations | 60-90 min |
| Day 3 | Batch CapCut edits + write captions + schedule publishing | 60 min |
| Day 4-7 | Content auto-publishes, monitor data, engage comments, prep next batch | Spare time |

## Batch Image Generation Checklist

For each batch of 10 videos:

```
[ ] 10 character references selected (Pinterest)
[ ] 10 base images generated (Nano Banana Pro)
[ ] Quality check: faces consistent, no AI artifacts
[ ] 10 "before" transformation images generated
[ ] 10 "after" transformation images generated
[ ] Quality check: same person in before/after pairs
[ ] All images saved with naming convention:
    batch_[date]_[number]_[type].png
    Example: batch_20260315_01_before.png
```

## Batch Video Generation Checklist

```
[ ] 10 motion references selected
[ ] Platform chosen for each video (Kling/Veo/Sora/Hailuo)
[ ] 10 videos generated (3 variants each → pick best)
[ ] Quality check: smooth transformation, no artifacts
[ ] All videos downloaded with naming convention:
    batch_[date]_[number]_video.mp4
```

## Batch Editing Checklist

```
[ ] 10 trending audios selected
[ ] 10 text overlays written (short, punchy, human)
[ ] CapCut template loaded
[ ] 10 videos edited:
    - Audio synced
    - Text overlays added
    - Quick cuts on beat
    - Duration: 10-15s
    - Natural ending (no logo, no CTA)
[ ] All exported: 9:16, 1080x1920, H.264, 30fps
```

## Publishing Schedule

```
[ ] 21 videos scheduled across 7 days (3/day)
[ ] Platforms: TikTok + Instagram Reels + YouTube Shorts
[ ] Staggered posting times:
    - Morning: 7-10 AM (target timezone)
    - Evening: 6-10 PM (target timezone)
[ ] Captions written for each platform
[ ] Hashtags researched and added
```

## Remake Protocol / 翻拍机制

When a video performs well, IMMEDIATELY create variants:

1. Same concept + different character
2. Same concept + different audio
3. Same concept + different caption angle
4. Same character + different transformation type
5. Same everything + different platform-native elements

One validated concept → 5-10 variants.
